# alx-low_level_programming

This repository contains the projects and tasks completed as part of the **Low-Level Programming** track at **ALX Software Engineering**. The primary focus is learning the **C programming language** from the ground up, moving from basic syntax to advanced systems-level concepts like memory management, data structures, and Unix system calls.

## 🛠️ Technical Stack
* **Language:** C (Standard: gnu89)
* **Style Guide:** [Betty Linter](https://github.com/alx-tools/Betty)
* **Environment:** Ubuntu 20.04 LTS
* **Compiler:** `gcc` (with `-Wall -Werror -Wextra -pedantic` flags)

## 📂 Repository Contents

| Directory | Topic | Description |
| :--- | :--- | :--- |
| `0x00-hello_world` | Basics | Introduction to C, compilation, and basic output (`printf`, `puts`). |
| `0x01-variables_if_else_while` | Logic | C data types, arithmetic, and control flow. |
| `0x02-functions_nested_loops` | Functions | Learning to modularize code and handle nested loops. |
| `0x03-debugging` | Troubleshooting | Manual debugging and understanding code logic errors. |
| `0x04-more_functions_nested_loops` | Functions | Advanced logic within function structures. |
| `0x05-pointers_arrays_strings` | Memory | Introduction to pointers, memory addresses, and array manipulation. |
| `0x06-pointers_arrays_strings` | Strings | Advanced string manipulation and custom string functions. |
| `0x07-pointers_arrays_strings` | Multi-dimensional | Multi-dimensional arrays and more pointer arithmetic. |
| `0x08-recursion` | Algorithms | Implementing recursive functions and managing the stack. |
| `0x09-static_libraries` | Libraries | Creating and using static libraries (`.a` files). |
| `0x0A-argc_argv` | Commands | Handling command-line arguments in C. |
| `0x0B-malloc_free` | Heap Memory | Dynamic memory allocation using `malloc` and `free`. |
| `0x0C-more_malloc_free` | Heap Memory | Advanced dynamic memory management. |
| `0x0D-preprocessor` | Macros | Working with macros and header files. |
| `0x0E-structures_typedef` | Data Structures | Defining custom data types with `struct` and `typedef`. |
| `0x0F-function_pointers` | Logic | Using pointers to call functions dynamically. |
| `0x10-variadic_functions` | Functions | Creating functions that take a variable number of arguments. |
| `0x12-singly_linked_lists` | Data Structures | Implementation of basic Singly Linked Lists. |
| `0x13-more_singly_linked_lists` | Data Structures | Advanced node manipulation in Singly Linked Lists. |
| `0x14-bit_manipulation` | Low-level | Working with bitwise operators (`&`, `|`, `^`, `<<`, `>>`). |
| `0x15-file_io` | Systems | Reading from and writing to files using system calls. |
| `0x17-doubly_linked_lists` | Data Structures | Implementation of Doubly Linked Lists (forward/backward traversal). |
| `0x18-dynamic_libraries` | Libraries | Creating and linking shared/dynamic libraries (`.so` files). |
| `0x1A-hash_tables` | Data Structures | Implementing Hash Tables and handling collisions. |
| `0x1C-makefiles` | Automation | Writing Makefiles to automate the compilation process. |
| `0x1E-search_algorithms` | Algorithms | Linear and Binary search implementations and time complexity analysis. |
