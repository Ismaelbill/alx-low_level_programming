# alx-low_level_programming

This repository contains the projects and tasks completed as part of the **Low-Level Programming** track at **ALX Software Engineering**. The primary focus is learning the **C programming language** from the ground up, moving from basic syntax to advanced systems-level concepts like memory management, data structures, and Unix system calls.

## 🛠️ Technical Stack
* **Language:** C (Standard: gnu89)
* **Style Guide:** [Betty Linter](https://github.com/alx-tools/Betty)
* **Environment:** Ubuntu 20.04 LTS
* **Compiler:** `gcc` (with `-Wall -Werror -Wextra -pedantic` flags)

## 📂 Project Structure
Each directory in this repository represents a specific module of the curriculum:

| Directory | Topic | Key Concepts |
| :--- | :--- | :--- |
| `0x00-hello_world` | Basics | Compilation process, `printf`, `puts` |
| `0x02-functions_nested_loops` | Logic | Functions, loops, header files |
| `0x05-pointers_arrays_strings` | Memory | Pointers, memory addresses, arrays |
| `0x08-recursion` | Algorithms | Recursive functions and stack management |
| `0x0E-structures_typedef` | Data | Defining custom types and structs |
| `0x11-printf` | Team Project | Replicating the standard `printf` function |
| `0x13-more_singly_linked_lists` | Structures | Dynamic memory, node manipulation |
| `0x16-simple_shell` | Systems | Forking, wait, execve, signal handling |
